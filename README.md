# PetCare Web Application

Aplikasi web untuk manajemen klinik hewan berbasis PHP dan MySQL.  
Fitur:
- Booking / Reservasi pemeriksaan hewan  
- Rawat inap  
- Kelola data hewan dan pemilik  
- Stok obat  
- Dashboard monitoring  
- Sistem pembayaran & invoice  

Dibuat sebagai Proyek UAS Pemrograman Web.
